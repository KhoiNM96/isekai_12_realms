# Privacy/Data Safety Draft

This is a planning draft, not legal advice.

- The game can be played offline.
- Local save data is stored on the player's device.
- Optional cloud save may be used if the player signs in and Firebase is configured.
- Firebase Auth may create an anonymous user ID for cloud save.
- Google Sign-In is optional if configured later.
- IAP purchase records may be stored locally and optionally synced for purchase recovery.
- The game does not include chat.
- The game does not include real-time multiplayer.
- The game does not include user-generated public content.
- The game does not collect precise location.
- The game does not collect contacts.
- The game does not use microphone or camera.
- The game does not include a direct ad SDK unless added later.
